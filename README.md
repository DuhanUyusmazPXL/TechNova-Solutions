# TechNova-Solutions

in deze repository bevat de cisco configuraties die gebruikt zijn voor het netwerkproject van TechNova Solutions

## Project Overzicht
- Netwerk ontworpen in cisco packet tracer
- Afdelingen: HR Ontwikkelaars, Gastennetwerk
- inter-VLAN routing

## VLANs
- VLAN 10 – HR (192.168.10.0/24)
- VLAN 20 – Ontwikkelaars (192.168.20.0/24)
- VLAN 30 – Gastennetwerk (192.168.30.0/24)

## bestanden
- `router-config.txt` – Cisco ISR-router configuratie
- `switch-config.txt` – Cisco 2960-switch configuratie
